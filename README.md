📩 AI News Automation Workflow (n8n)

This project is an automated AI + Technology news summarizer built using n8n. It fetches the latest news from RSS feeds, processes the content using an LLM (Google Gemini), and sends a structured summary directly to your email.

🚀 Overview

This workflow performs the following steps automatically:

Fetches news from multiple RSS feeds
Merges and aggregates the data
Uses an AI model to summarize and categorize the news
Sends a clean, formatted email with the summary
⚙️ Workflow Architecture
Schedule Trigger → RSS Feeds → Merge → Aggregate → LLM (Gemini) → Gmail भेजना
🧩 Nodes Explanation
1. ⏰ Schedule Trigger
Triggers the workflow at defined intervals.
Can be configured to run daily, hourly, etc.
2. 📰 RSS Feed Readers
RSS Read (AI Business)
Source: https://aibusiness.com/rss.xml
Fetches AI-related news
RSS Read1 (TechCrunch)
Source: https://techcrunch.com/feed/
Fetches general technology news
3. 🔀 Merge Node
Combines data from both RSS feeds into a single stream.
4. 📦 Aggregate Node
Collects all news items into one structured dataset.
Prepares data for AI processing.
5. 🤖 LLM Processing (Google Gemini)
Uses Google Gemini Chat Model to:
Analyze the aggregated news
Categorize into:
AI News
Technology Updates
Generate summaries (1–2 lines each)
Format output in a structured email style
6. 📧 Gmail Node
Sends the final summarized news to your email.
Includes:
Subject: Today's AI News for you...
Clean, readable content generated by the LLM
✨ Output Format

The email will look like:

Hi there,
Here's your Tech Brief:

Summary
Link

HEADLINE
